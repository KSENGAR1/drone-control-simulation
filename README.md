This project implements a multi-axis quadcopter control system using PID controllers and visualizes the drone behavior in 3D.

---

## 🔧 Features

- Altitude control using closed-loop PID system  
- Roll and pitch stabilization  
- Disturbance response analysis  
- 3D drone visualization using MATLAB  

---

## 🛠️ Tools Used

- MATLAB  
- Simulink
